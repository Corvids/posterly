# Posterly

Deployed at https://posterly-nine.vercel.app/

## Getting Started

### Required

- Node v18+ 
- npm

### Local development

1. Install dependencies:
```bash
npm install
```

2. Run the dev server:
```bash
npm run dev
```

4. Open [http://localhost:3000](http://localhost:3000) in your favorite browser
